# Java XML Processing Laboratory Work 3

## Опис

Проєкт реалізує обробку XML документа з використанням SAX parser, DOM parser та XSD validation у Java.

У якості джерела даних використовується датасет Popular Baby Names NY, що містить інформацію про популярні дитячі імена у місті Нью-Йорк.

Програма дозволяє:

* аналізувати XML документ;
* отримувати список XML тегів;
* отримувати етнічні групи;
* вибирати найбільш популярні імена;
* створювати Java об’єкти;
* сортувати дані за рейтингом;
* створювати новий XML документ;
* читати XML документ;
* виконувати XSD validation.

# Реалізовані можливості

## SAX Parser

Реалізовано:

* читання XML документа;
* отримання списку тегів;
* аналіз структури документа;
* отримання етнічних груп;
* вибір популярних імен.

---

## DOM Parser

Реалізовано:

* створення нового XML документа;
* запис інформації до XML;
* читання XML документа.

---

## XSD Validation

Реалізовано:

* створення XSD схеми;
* перевірка XML документа;
* валідація структури XML.

---

# Логіка роботи програми

1. Програма читає XML документ через SAX parser.
2. Отримується список XML тегів.
3. Отримуються етнічні групи.
4. Вибираються найбільш популярні імена.
5. Створюються Java об’єкти.
6. Дані сортуються за рейтингом.
7. Створюється новий XML документ.
8. Новий XML документ читається через DOM parser.
9. Виконується XSD validation.

# Примітка

У проєкті використовується великий XML датасет Popular Baby Names NY.

Під час виконання роботи було виявлено, що реальні XML датасети можуть містити:

* вкладені структури;
* службові attributes;
* скорочені або неповні дані.

Для коректної роботи XSD validation була створена адаптована XML схема.
